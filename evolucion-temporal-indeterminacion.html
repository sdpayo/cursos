Evolución Temporal del Sonido y Principio de Indeterminación

Tiempo y frecuencia

Envolvente de Sonido (ADSR)

La evolución temporal de un sonido define su carácter dinámico. El modelo clásico ADSR describe:

  • Attack (A): Tiempo para alcanzar el nivel máximo desde el inicio.
  • Decay (D): Descenso desde el pico al nivel sostenido.
  • Sustain (S): Nivel mantenido mientras suena la nota.
  • Release (R): Tiempo para caer a cero tras soltar la nota.

Principio de Indeterminación de Gabor

Precisión Temporal vs Frecuencial

No es posible especificar simultáneamente el tiempo exacto y la frecuencia exacta de un evento sonoro.

Trade-off Fundamental

Mayor precisión temporal = menor precisión frecuencial, y viceversa.

Análisis de Fourier de Corta Duración

La STFT (Short-Time Fourier Transform) representa este compromiso equilibrado.

Implicaciones Prácticas

  • Síntesis: Manipulación de envolventes para control expresivo.
  • Análisis: Elección del tamaño de ventana en procesamientos.
  • Transientes: Sonidos cortos (clicks, percusión) requieren análisis temporal preciso.
  • Tonos Sostenidos: Mejor análisis frecuencial con ventanas largas.
  • Compresión de Audio: Distribución de bits según el análisis tiempo-frecuencia.
